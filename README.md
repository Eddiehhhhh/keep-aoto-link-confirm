# Notion 自动关联

自动关联 Notion 的"运动"数据库和"日记中心"数据库中日期相同的项。

## 功能

- 每天北京时间 23:00 自动运行（通过 GitHub Actions）
- 查询近 7 天的运动记录和日记记录
- 自动匹配日期相同的记录
- 更新双向关联属性

## 配置

### 数据库 ID（已内置，无需修改）

- 运动数据库: `26333b33-7f23-81ed-b436-000b62507748`
- 日记中心: `6d280b25-c22e-432a-a8e9-22e72617205e`

### 属性映射

| 运动数据库 | 日记中心 |
|-----------|---------|
| 开始时间 (日期) | 日期 (日期) |
| 📅 日记中心 (关联) | 运动 (关联) |

## 技术细节

- **API 版本**: Notion API v2025-09-03
- **查询范围**: 近 7 天
- **运行频率**: 每天 UTC 15:00 (北京时间 23:00)
