# Keep 自动关联

自动关联 Keep 运动记录与 Notion 日记中心，日期相同的自动关联。

## 功能

- 每天北京时间 23:00 自动运行（GitHub Actions）
- 查询近 7 天的运动记录和日记记录
- 自动匹配日期相同的记录
- 更新双向关联属性

## 配置

| 数据库 | ID |
|--------|-----|
| Keep 运动数据库 | `26333b33-7f23-81ed-b436-000b62507748` |
| 日记中心 | `6d280b25-c22e-432a-a8e9-22e72617205e` |

## 属性映射

| Keep | 日记中心 |
|------|---------|
| 开始时间 (日期) | 日期 (日期) |
| 📅 日记中心 (关联) | 运动 (关联) |
