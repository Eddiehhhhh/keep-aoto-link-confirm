# Notion 自动关联脚本

自动关联 Notion 的"运动"数据库和"日记中心"数据库中日期相同的项。

## 功能

- 每天自动运行（通过 GitHub Actions）
- 查询近 7 天的运动记录和日记记录
- 自动匹配日期相同的记录
- 更新双向关联属性

## 快速开始

### 1. 创建 GitHub 仓库

在 GitHub 上创建一个新的私有仓库（建议命名为 `notion-auto-link`），然后：

```bash
cd notion-auto-link
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/notion-auto-link.git
git push -u origin main
```

### 2. 配置 Secrets

在 GitHub 仓库的 **Settings → Secrets and variables → Actions** 中添加：

| Name | Value |
|------|-------|
| `NOTION_TOKEN` | 你的 Notion API Token |

获取 Token 方法：
1. 访问 [Notion Integrations](https://www.notion.so/my-integrations)
2. 点击 "New integration"
3. 创建后复制 Token

### 3. 授权 Integration 访问数据库

1. 打开你的 Notion 数据库页面
2. 点击右上角 **...** → **Add connections**
3. 搜索并添加你创建的 Integration

### 4. 完成！

Actions 会自动在每天 23:00 (北京时间) 运行。

## 本地测试

```bash
# 设置环境变量
export NOTION_TOKEN="your_token_here"

# 运行脚本
python3 main.py
```

## 手动触发

在 GitHub 仓库的 **Actions** 页面，点击 "Run workflow" 可以手动触发。

## 技术细节

- **API 版本**: Notion API v2025-09-03
- **查询范围**: 近 7 天
- **关联字段**: 
  - 运动数据库: "📅 日记中心"
  - 日记中心: "运动"
- **日期匹配**: 运动"开始时间" ↔ 日记"日期"
